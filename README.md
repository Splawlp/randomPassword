# [Random Password Generator](https://randomizerforpasswords.netlify.app/) 🔐

A simple and clean web app for generating random, secure passwords. Built with **HTML**, **CSS**, and **Vanilla JavaScript**.

## ✨ Features

- Generate from **1 to 10** passwords at once
- Customizable password length (**8–20** characters)
- Optional inclusion of **numbers** and **special characters**
- **Copy to clipboard** with one click (visual confirmation icon)
- **Light/Dark theme** switcher with preference saved in `localStorage`
- Responsive, minimalist UI based on a custom Figma design

## 🛠️ Tech Stack

- HTML5
- CSS3 (custom styling, Karla font from Google Fonts)
- JavaScript (DOM manipulation, Clipboard API, localStorage)

## 🎨 Design & Inspiration

- **UI/UX layout** — designed in [Figma](https://www.figma.com/file/NEj9JDycMjF3XKXq7swoc9/Random-Password-Generator-(New-version)?node-id=0%3A1)
- **Theme toggle button** — inspired by the implementation from [coding2go.com](https://coding2go.com/)
- Got help from ChatGPT to better understand how to work with **checkboxes** and **input fields** in the password options form

## 🚀 How It Works

1. Choose whether to include numbers and/or special characters using the checkboxes
2. Set your desired password length (8–20)
3. Choose how many passwords to generate (1–10)
4. Click **"Generate passwords"**
5. Click on any generated password to copy it to your clipboard

# Генератор випадкових паролів 🔐

Простий та зручний веб-застосунок для генерації випадкових надійних паролів. Створений на **HTML**, **CSS** та **Vanilla JavaScript**.

## ✨ Можливості

- Генерація від **1 до 10** паролів одночасно
- Налаштування довжини пароля (**від 8 до 20** символів)
- Можливість додавання **цифр** та **спецсимволів**
- **Копіювання в буфер обміну** одним кліком (з візуальним підтвердженням)
- Перемикач **світлої/темної теми** зі збереженням вибору в `localStorage`
- Адаптивний мінімалістичний інтерфейс на основі власного макета у Figma

## 🛠️ Технології

- HTML5
- CSS3 (власні стилі, шрифт Karla з Google Fonts)
- JavaScript (робота з DOM, Clipboard API, localStorage)

## 🎨 Дизайн та джерела натхнення

- **Макет інтерфейсу** — розроблений у [Figma](https://www.figma.com/file/NEj9JDycMjF3XKXq7swoc9/Random-Password-Generator-(New-version)?node-id=0%3A1)
- **Кнопка перемикання теми** — запозичена ідея з [coding2go.com](https://coding2go.com/)
- У роботі з **чекбоксами** та **полями введення** для форми налаштувань пароля допоміг розібратись ChatGPT

## 🚀 Як це працює

1. Обери, чи додавати цифри та/або спецсимволи за допомогою чекбоксів
2. Встанови потрібну довжину пароля (8–20)
3. Вкажи кількість паролів для генерації (1–10)
4. Натисни **"Generate passwords"**
5. Клікни на будь-який згенерований пароль, щоб скопіювати його в буфер обміну
